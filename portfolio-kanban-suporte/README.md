# Kanban Suporte

Quadro de atendimento em React para simular uma fila de suporte B2B. Ele foi pensado como projeto de portfolio por ter fluxo de produto claro, estado mutavel e filtros simples.

## O que o projeto demonstra

- Busca por ticket, cliente ou assunto.
- Filtro por prioridade.
- Movimento de cards entre colunas.
- Contadores por etapa e indicadores de SLA.

## Como rodar

```bash
npm install
npm run dev
```
