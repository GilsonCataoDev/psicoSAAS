# Financeiro SaaS

Dashboard React para acompanhamento financeiro de uma operacao SaaS B2B. O projeto evita backend de proposito: os dados estao no front para facilitar apresentacao em entrevista e discutir como a tela seria conectada a uma API.

## O que o projeto demonstra

- Estado local para filtro de faturas.
- Metricas derivadas dos dados.
- Grafico simples feito com CSS.
- Layout responsivo para desktop e mobile.

## Como rodar

```bash
npm install
npm run dev
```
