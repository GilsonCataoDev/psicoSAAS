# Agenda Medica

Painel em React para acompanhamento de agenda diaria em uma clinica. O foco e mostrar filtros, composicao de componentes, estado local e organizacao visual sem depender de API externa.

## O que o projeto demonstra

- Busca por paciente, profissional ou observacao.
- Filtro por status da consulta.
- Cards responsivos com informacoes operacionais.
- Metricas laterais calculadas a partir dos dados.

## Como rodar

```bash
npm install
npm run dev
```
